<h2 align="center"> BIKE STORE APP</h2>
<p align="center">A react app mockup created for a bike store</p>

### Development
Tools Used:
* ReactJS
* Webpack
* Babel
* Uikit

To install and run locally:

```sh
$ git clone https://github.com/singhrohan62/bike-store
$ npm i && npm run dev
```
